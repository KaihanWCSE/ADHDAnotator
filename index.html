About this demo

ADHD Anotator turns dense PDF paragraphs into short, clickable bullet points so the main ideas are easier to scan. Each bullet links back to the exact original text it came from, so the summary stays connected to the source instead of replacing context.

This version is running in mock mode for the hackathon demo. The PDF, bullet structure, and source-text popovers are working, but the AI summarization step is temporarily using a fixed mock response because Butterbase AI encountered account error during development and could not give me any AI credits.

The current demo auto-loads a test PDF so reviewers can see the intended reading experience end to end. The next step is to connect the same workflow to a live LLM API so uploaded PDFs can be summarized dynamically.
